### スキル2: キーワードプール管理SQL

```sql
-- 未使用キーワード在庫確認
SELECT category, COUNT(*) as unused_count
FROM blog_keywords
WHERE used = false
GROUP BY category
ORDER BY unused_count ASC;

-- キーワード追加
INSERT INTO blog_keywords (keyword, category, used) VALUES
('キーワード1', 'category', false),
('キーワード2', 'category', false);

-- 在庫が5個以下のカテゴリを警告
SELECT category, COUNT(*) as remaining
FROM blog_keywords
WHERE used = false
GROUP BY category
HAVING COUNT(*) <= 5
ORDER BY remaining ASC;
```

## ⛔ やってはいけないこと

- ビッグキーワード（月間10万以上）を狙う
- 検索ボリュームゼロのキーワードを大量投入
- カテゴリの偏り（1カテゴリに集中）
- 季節外れキーワードの投入
- 競合が強すぎるキーワードへの挑戦
