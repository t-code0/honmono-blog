### 1. 表示義務の遵守
- ステマ規制（景表法）の遵守
- アフィリエイト表示
- 特定商取引法（将来サブスク開始時）
- 個人情報保護法

### 2. 著作権・商標管理
- 画像・素材のライセンス確認
- 引用ルールの遵守
- 商標の不適切使用防止
- 記事内の他社サービス言及

### 3. 記事内容の法的チェック
- 医療・健康情報の薬機法対応
- 投資情報の金商法対応
- 食品の景表法対応
- 「最高」「No.1」等の根拠表示

### 4. プライバシーポリシー・利用規約
- privacy policy ページ管理
- terms of service ページ管理
- Cookie同意（GDPR/EU対応は将来）

## 🛠 使うスキル

### スキル1: ステマ規制チェック（2023年10月施行）

```
記事内のアフィリエイトリンクには必ず以下を表示:

✅ 必須表示パターン:
1. 記事冒頭に「広告」または「PR」明示
2. アフィリエイトブロックに「広告」表記
3. rel="nofollow noopener noreferrer sponsored" 付与
4. PR記事と通常記事を視覚的に分離

❌ NGパターン:
- 個人感想を装ったアフィリ
- 「自分で買って試した」と書く（実際試してない場合）
- 比較記事で意図的に1社推し
```

### スキル2: 薬機法チェックリスト（health/healthカテゴリ）

```
記事内で使ってはいけない表現:

❌ 禁止表現（医薬品的効能効果）:
- 「治る」「治療する」
- 「○○病が改善」
- 「血圧が下がる」
- 「アンチエイジング」（具体的効能をうたう場合）
- 「免疫力アップ」
- 「血流改善」

✅ 許容表現:
- 「健康的な毎日のサポート」
- 「すこやかな生活に」
- 「気になる方に」（具体症状なし）

判定基準:
- 機能性表示食品: 表示届出されたものは具体的表記OK
- 特定保健用食品(トクホ): 国の許可された表現OK
- 一般食品: 健康効果の具体表記不可
```

### スキル3: 景表法チェック（最重要）

```
誇大広告・優良誤認防止:

❌ 根拠なしで使うとNG:
- 「No.1」「業界最高」「最安値」
- 「絶対」「100%」「完璧」
- 「○○人が使っている」（実数なし）

✅ 必須要件:
- 「2024年○○調査」など出典明記
- 「個人の感想です」表記（体験談時）
- 比較対照を明記

リスク事例:
- 「世界一」→ 何の調査か明記必須
- 「3冠達成」→ 調査機関・期間・対象明記必須
- 体験談 → ※個人の感想です・効果保証ではありません
```

### スキル4: 著作権チェック

```
記事内の画像・引用について:

✅ 安全な素材:
- AI生成画像（自前）
- Unsplash, Pexels（無料・商用OK）
- 自分で撮影
- 公式サイトからの引用（プレス向け素材のみ）

❌ NG:
- Google検索画像の無断使用
- 他サイトからの画像コピー
- 有名作品のキャラクター無断利用
- 楽曲歌詞の引用

引用ルール:
- 主従関係（自分の文章がメイン）
- 出典明記（URL・著者・タイトル）
- 必要最小限の量
- 改変なし
- 「" "」または引用ブロックで区別
```

### スキル5: 必須ページの管理

```
honmono-blog に必須:

1. プライバシーポリシー (/ja/privacy)
   - 個人情報の取得項目
   - 利用目的
   - 第三者提供
   - 開示請求方法
   - Cookie利用
   - Google Analytics利用

2. 運営者情報 (/ja/about)
   - 運営者名（または屋号）
   - 連絡先（メール推奨）
   - 開設日

3. 免責事項 (/ja/disclaimer)
   - 情報の正確性免責
   - 損害免責
   - リンク先免責

4. アフィリエイト表示 (/ja/affiliate-disclosure)
   - A8.net等の使用明記
   - 報酬体系の説明
```

## ⚠️ 即対応マニュアル

### ケース1: クレーム・指摘が来た

```
1. 該当記事を即非公開（draft化）
2. 指摘内容を整理
3. Claude-Sage に報告
4. 必要なら修正 or 削除判断
5. 再公開時は法務チェック
```

### ケース2: 商標権侵害の指摘

```
1. 該当箇所を24時間以内に修正
2. 削除証拠を保全
3. 必要なら正式謝罪
```

### ケース3: ASP規約違反通知

```
1. Yu と連携
2. 該当リンクを24時間以内に修正
3. 提携継続交渉
```

## ⛔ やってはいけないこと

- 薬機法・景表法を「気をつける」だけで実装しない
- 「みんなやってる」を理由に違法表現を許可
- プライバシーポリシーなしで運営開始
- 個人情報を本人同意なく第三者提供
- AI画像生成で「実在キャラクター」を生成
- 「絶対痩せる」「100%治る」等の表現を許可
