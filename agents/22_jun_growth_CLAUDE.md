### スキル3: ファネル分析

```sql
-- ファネル分析クエリ

-- Step 1: 流入（記事閲覧数）
SELECT category, SUM(view_count) as views
FROM blog_articles
WHERE status = 'published'
GROUP BY category;

-- Step 2: 回遊（1ユーザーあたり閲覧ページ数）
-- GA4連携後に実装

-- Step 3: アフィリクリック
SELECT
  ba.category,
  SUM(ba.view_count) as views,
  COUNT(bac.id) as clicks,
  ROUND(COUNT(bac.id)::numeric / NULLIF(SUM(ba.view_count), 0) * 100, 3) as click_rate
FROM blog_articles ba
LEFT JOIN blog_affiliate_clicks bac ON bac.article_id = ba.id
WHERE ba.status = 'published'
GROUP BY ba.category
ORDER BY click_rate DESC;

-- Step 4: 成果（CVR）
-- ASPレポートと突合（手動）
```
